# Question 2 - Periodic Table

This project is a console-based application to store and access the first 30 elements of the periodic table.

## Features

- Store first 30 elements using Dictionary
- Search element using atomic number
- Display element details (Atomic Number, Name, Type)
- Repeat search using do while loop
