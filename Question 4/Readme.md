# Question 4 - Web Scraping and CSV

This program fetches data from a website,extracts book details,stores them in a CSV file and reads the data back.

---

## Features

- Fetches webpage using requests

- Extract book details using BeautifulSoup

- Extract Name, Price, and Rating

- Store data in CSV file

- Read data from CSV file

- Display data in terminal

---

## Files

- Que4.py - Main program

- books.CSV - Stores data

- Screenshots - Output images
