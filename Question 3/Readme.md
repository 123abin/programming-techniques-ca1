# Question 3 - Client Server System

This project is a simple client server application using Python.

The client sends customer details to the server.
The server stores the data in a database and sends back a registration number.

---

## Features

- Send customer details from client
- Receive data on server
- Store data in SQLite database
- Generated the registration number
- Send registration number back to client

---

## Files

- Que3-client.py - Client program
- Que3-server.py - Server program
- customers.db - Database
- Send registration number back to client

---
